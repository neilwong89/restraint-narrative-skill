# 逐句写作指导

> 详细版，写这篇时参考。对应 SKILL.md 第五步"正文写作"。

---

## 开头写法

### 推荐开场方式（按优先级）

**① 消息触发型（最常用）**
```
他发来消息：「你也这么晚？」
```
开场直接进入对话，留白感最强。

**② 朋友圈触发型**
```
灯关了。手机屏幕还亮着。
她发了一条朋友圈，没有文案，只有一张照片。
```
用"没有文案"暗示她有话说不出口。

**③ 时间地点型**
```
那年冬天，办公室的暖气坏了。
```
不要写"有一天"，季节锚点比"有一天"更有质感。

### 绝对禁止的开头

- ❌ "首先……"/"随着……"/"大家好……"
- ❌ "在我二十多岁的时候……"
- ❌ "我有一个朋友……"
- ❌ "今天给大家讲一个故事……"

## 对话写法

### 格式规范

```markdown
他：「消息内容」
我：「消息内容」
他：<发来一张照片，没有配文>
```

每条消息单独成段，用「」包裹。消息之间的叙事夹叙，每段不超过10字。

### 消息特征

微信消息是不完整的：
- 没主语：「你也这么晚？」
- 只有一个字：「嗯」「好」「在」
- 没有声音：发来一首歌，不说话
- 反复撤回又发：「消息已撤回」「消息已撤回」

### 叙事衔接词库

不用"然后"，用以下词自然衔接：

| 场景 | 用词 |
|------|------|
| 对方没回 | 我把手机放下了。没再追问。 |
| 想回又删 | 我打了几个字，删了。又打了一行，删了。 |
| 对方在线 | 他的头像亮了。我没有点开。 |
| 深夜独处 | 灯关了。手机屏幕还亮着。 |
| 动作中断 | 他停了一下，说，没什么。 |
| 消息列表 | 我把那条消息往上划，又划回来。 |
| 发完后悔 | 我发完就后悔了。但没有撤回。 |

## 意象的使用

### 三类意象

**场景意象**：月亮/夜色/路灯/雨/车窗/办公室窗/停车场
- callback 写法：「今晚的月亮很圆。」（消息形式）
- 或：雨还在下。他把伞收了。

**物品意象**：外套/雨伞/本子/照片/成绩单/便签
- callback 写法：「她从柜子里翻出那本本子，拍了拍灰。」
- 不通过对话说出，用动作重现。

**行为意象**：发了又撤/存下来/晚安/微信运动步数
- callback 写法：「她把那条语音又听了一遍。」

### 意象的度

意象是留白工具，不是议论工具：
- ❌ 「那轮月亮，就像他对我说的那句晚安」
- ❌ 她想起他说过的话，就像那轮月亮
- ✅ 「他把那张照片发过来。我看了很久，没有说话。」

## 言行矛盾写法

### 操作步骤

**第一步：建立表面言行一致**

先写他说的公事话，让读者觉得这就是正常的上下级/同事关系。

**第二步：埋入矛盾**

让他做一件和说的话矛盾的事。读者读到那里会愣一下——他说的是A，但他做的是B。

**第三步：不点破**

作者不解释，让矛盾自己暴露。读者自己发现的那一刻，比任何情绪词都有力。

### 正确示范

```
他：「以后有事直接说，不用绕弯子。」
她：「好。」
（第二天，他发来消息：「今天食堂有你爱吃的糖醋排骨。」）
```

```
他：「我这个人不会说话。」
她：「嗯。」
（但他的手机备忘录里，记着她说过的每一句话。）
```

### 禁止示范

```
他嘴上说不要联系，却一直在发消息。  ← 这是议论，不是场景
其实他心里是在乎我的。               ← 议论
她明白他的意思了。                    ← 议论
```

## 转折写法

### 转折必须用具体事件承载

不能写「后来我心态变了」这种空洞转折。转折必须是一个：
- 具体场景（停车场看见他的车没开灯）
- 具体对话（「你知道我为什么今晚回来吗」）
- 具体动作（他把东西装进纸箱，在工位前停了三秒）

### 车里场景写法

车里是此类文章最经典的转折场景，因为：
- 封闭空间，两个人被迫独处
- 可以不说话，只写动作
- 开车/停车是天然的"走/留"隐喻

```
车停在楼下，他没有熄火。
我解开安全带，说谢谢领导。
他手放在方向盘上，没有动。
过了很久，他说了一句话。
```

## 结尾写法

### 四种可选结尾

**① 消息戛然而止型**
最后一条消息发出，对方没有再回。故事结束。

**② 意象留白型**
以场景描写收尾，不解释。

**③ 动作收尾型**
以一个人的动作为结尾，他没有说话。

**④ 回调意象型**
开头出现的意象在结尾再次出现，但不解释。

### 结尾禁止词

- ❌ "这个故事告诉我们……"
- ❌ "愿你……"/"愿所有……"
- ❌ "所以我们……"
- ❌ "其实……"（总结性的）
- ❌ "根据网友倾诉撰写"

### 结尾高潮句要短

全篇最有力的一句话，通常不超过15字：

```
"我女儿。"（3字）
"就是习惯了。"（4字）
"他发来最后一条：晚安。"（10字）
"灯还亮着。"（4字）
```
