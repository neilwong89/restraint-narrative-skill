# 克制情感叙事文 · 实例解析

> 从两篇测试文章中提取的写作技法与质检逻辑。正文均通过四层质检，可作为同类创作的参考范本。

---

## 实例一：「导师」

**素材：** 女研究生 + 男导师，外校联培错过，无表白，结尾回调空调/窗外照片/批注

**通过的写法**

| 技法 | 原文 | 说明 |
|------|------|------|
| 场景代替情绪 | "我把那张照片翻出来，盯了很久，给他发了过去" | 没有"我很难过"，只有动作 |
| 对话残缺 | 他：「睡不着」 | 凌晨两点批注改成了"睡不着"，不说破 |
| 言行矛盾 | 他说"出差不能送"，但问"到了告诉我一声" | 嘴上说忙，行为是习惯性关心 |
| 物品意象callback | 开头"空调坏了"→结尾"空调后来还坏过吗""有时" | 场景意象回环，不解释 |
| 照片意象callback | 开头发窗外夜景→结尾回发同一张照片 | 物品意象callback，"不知道为什么要存" |

**修复记录：** 原文有"我想了很久，给他回了一张照片"→改为"我把那张照片翻出来，盯了很久，给他发了过去"。去掉"我想"，保留动作。

## 实例二：「食堂的糖醋排骨」

**素材：** 丧偶女同事 + 已婚男同事，工伤去世，微信互动，言行矛盾是核心张力

**通过的写法**

| 技法 | 原文 | 说明 |
|------|------|------|
| 言行矛盾（核心） | "他说让我以后不要联系了，但……" | 他嘴上推开，行为继续关心 |
| 言行矛盾高阶 | 他说"发错了"（语音），但那是孩子特意要发的祝福 | 嘴上说失误，实际暴露了他在孩子面前保留了她的存在 |
| 物品意象callback | "本子"出现2次，"明年带老婆去云南"是核心 | 小姜遗物→催泪结尾 |
| 行为意象callback | 她把语音"又听了一遍"，存下来 | 不用对话说情感，用动作承载 |
| 场景意象callback | 路灯出现3次：雨里亮着→雨里的路灯→那盏路灯 | 同一盏路灯，贯穿始终 |
| 糖醋排骨回环 | "很久不吃了"→"今天食堂有"→爱吃 | 食物意象串起日常与怀念 |

**言行矛盾技法解析：**

```
他：[语音] "姜叔叔新年快乐！"（孩子的声音）
他：把那条语音撤回了
他：发来文字：「不好意思，发错了。」

→ 矛盾：他说是失误，但那条语音是他让孩子录的、他主动发的
→ 她的反应：把语音存了下来，又听了一遍
→ 作者没有说"他其实很想联系我"，但读者自己懂了
```

## 常见错误对照

| 错误类型 | 错误写法 | 正确写法 |
|----------|----------|----------|
| 情绪形容词暴露 | "他发来消息，我很紧张" | "消息弹出来，我盯着屏幕看了三秒" |
| 作者议论 | "那一刻我知道，他是喜欢我的" | "他把车窗摇上去，开走了" |
| 对话太完整 | 他：「我给你发这条消息是因为我一直在想你」 | 他：「睡不着。」 |
| 结尾升华 | "这个故事告诉我们，有些感情注定没有结果" | "雨里的路灯，亮着。" |
| 意象用议论回收 | "那轮月亮，就像他对我说的那句晚安" | "她把那条消息往上划，划到那张照片" |
| 言行一致（无矛盾） | 他说不要联系，然后真的不联系了 | 他说不要联系，但还继续发"带伞了吗" |
