# 克制情感叙事文写作 skill

> 面向"抖音故事号"/"微信对话故事"赛道的 AI 写作 skill。核心风格：**让读者自己哭，自己心疼，自己脑补，作者一个字都不说。**

## 文件说明

```
restraint-narrative/
├── SKILL.md                     ← 主 skill 文件（Hermes Agent 直接加载）
└── references/
    └── examples.md              ← 实例解析 + 技法拆解
```

## 使用方法

在 Hermes Agent 中加载：

```
/skill restraint-narrative
```

或让 Agent 读取 `SKILL.md` 并按其中第六步"四层质检"执行。

## 核心规则（速查）

| 规则 | 说明 |
|------|------|
| 禁情绪词 | 紧张/心慌/感动/喜欢/爱 全部禁用 |
| 禁议论句 | 我想/我觉得/我知道 禁用于旁白 |
| 对话要短 | 「嗯」「好」「在吗」比长句更有力 |
| 意象callback | 开头出现的意象，结尾必须再次出现 |
| 言行矛盾 | 说的和做的不一致，是最强的张力 |
| 结尾禁止 | 不许总结、不许升华、不许"愿你" |

## 质检流程

写完必须跑四层质检：

1. **L1 硬性规则** — 情绪词零容忍
2. **L2 情感克制** — 有无言行矛盾
3. **L3 逻辑一致性** — 生死状态链/时间线/人物行为
4. **L4 意象callback** — 开头意象在结尾是否回调
5. **L5 错别字** — 的地得/一起提前/多字少字
6. **L6 阅读体验** — 有无跳过的冲动

## 平台调性

| 平台 | 字数 | 风格 |
|------|------|------|
| 抖音故事号 | 1500-2500 | 最克制，留白最多 |
| 微信公众号 | 2000-3000 | 可略多心理描写 |
| 知乎 | 1000-2000 | 可有少量议论 |
